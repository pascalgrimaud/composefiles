# How to use

Pull the image:
- `docker pull jhipster/jhipster-sample-app-gateway`

Start the gateway:
- `docker-compose up`

Start the gateway in background
- `docker-compose up -d`

Stop the gateway:
- `docker-compose stop`

Access to the gateway:
- http://localhost:8088
