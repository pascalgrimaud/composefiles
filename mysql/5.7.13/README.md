# How to use

Pull the image:
- `docker pull mysql:5.7.13`

Start MySQL:
- `docker-compose up`

Start MySQL in background
- `docker-compose up -d`

Stop MySQL:
- `docker-compose stop`

All data are under the volume:
- `~/volumes/mysql`
