# How to use

Pull the image:
- `docker pull postgres:9.5.4`

Start PostgreSQL:
- `docker-compose up`

Start PostgreSQL in background
- `docker-compose up -d`

Stop PostgreSQL:
- `docker-compose stop`

All data are under the volume:
- `~/volumes/postgresql`
